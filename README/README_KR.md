**Kemono Downloader**에 오신 것을 환영합니다. PyQt6로 구축된 다목적 Python 기반 데스크톱 애플리케이션으로, [Kemono.cr](https://kemono.cr)에서 콘텐츠를 다운로드하도록 설계되었습니다. 이 도구를 통해 사용자는 Patreon, Fanbox 등의 서비스에서 개별 게시물 또는 전체 크리에이터 프로필을 보관할 수 있으며, 사용자 지정 가능한 설정과 고급 기능을 갖춘 다양한 파일 유형을 지원합니다.

## 중요 안내사항

### 면책조항

KemonoDownloader는 개인 및 교육 목적으로만 설계된 도구로, 사용자가 Kemono.cr에서 콘텐츠를 다운로드하는 것을 돕습니다. 이 프로젝트의 관리자들은 **저작권으로 보호되는 자료의 무단 배포를 용인하거나 지지하지 않습니다**. 사용자는 Kemono.cr에서 콘텐츠에 액세스하고 다운로드할 법적 권리가 있는지 확인하고, 모든 적용 법률과 원본 플랫폼(예: Patreon, Pixiv Fanbox, Gumroad)의 서비스 약관을 준수할 단독 책임을 집니다.

**이 도구를 크리에이터의 권리 침해, 저작권법 위반, 또는 서비스 약관 위반에 오용하는 것은 엄격히 금지됩니다.** 관리자들은 법적 조치, 금전적 손실, 또는 제3자에 대한 손해를 포함하되 이에 국한되지 않는 KemonoDownloader의 오용이나 사용으로 인해 발생하는 결과에 대해 책임지지 않습니다.

### 윤리적 사용 가이드라인

KemonoDownloader는 Patreon, Pixiv Fanbox, Gumroad와 같은 유료 플랫폼에 원래 게시되었을 수 있는 자료를 포함하는 Kemono.cr의 콘텐츠와 상호작용합니다. 이러한 플랫폼의 많은 크리에이터들은 생계를 위해 유료 구독에 의존하고 있습니다. 허락 없이 그들의 콘텐츠를 다운로드하고 재배포하는 것은 계속 창작할 수 있는 능력을 해칠 수 있습니다.

다음 사항을 강력히 권장합니다:
- KemonoDownloader를 책임감 있게 사용하고, 법적으로 액세스할 권리가 있는 콘텐츠에만 사용하세요.
- Patreon, Pixiv Fanbox, 또는 Gumroad와 같은 플랫폼에서 크리에이터의 공식 채널을 구독하여 크리에이터를 직접 지원하세요.
- 다운로드한 콘텐츠의 재배포를 피하세요. 이는 저작권법을 위반하고 크리에이터에게 해를 끼칠 수 있습니다.

### 위험성 및 제한사항

- **법적 위험**: Kemono.cr에서 콘텐츠를 다운로드하는 것은 저작권법이나 원본 플랫폼의 서비스 약관을 위반할 수 있습니다. 사용자는 이 도구 사용과 관련된 모든 법적 위험을 부담합니다.
- **Kemono.cr 의존성**: KemonoDownloader는 일관성 없는 업데이트와 다운타임 이력이 있는 Kemono.cr에 의존합니다. Kemono.cr이 사용할 수 없게 되면 이 도구는 기능을 잃게 됩니다.
- **속도 제한 및 오류**: Kemono.cr은 다운로드 성능에 영향을 미치는 속도 제한이나 기타 제한을 부과할 수 있습니다. 관리자들은 Kemono.cr의 콘텐츠에 대한 중단 없는 액세스를 보장할 수 없습니다.

## 기능
- **게시물 다운로드**: URL을 사용하여 특정 Kemono.cr 게시물에서 파일을 쉽게 다운로드.
- **크리에이터 아카이브**: 한 번의 클릭으로 크리에이터 프로필에서 모든 게시물과 파일을 일괄 다운로드.
- **파일 유형 지원**: 이미지(JPG, PNG, GIF), 비디오(MP4), 아카이브(ZIP, 7Z), PDF 등을 처리.
- **동시 다운로드**: 최적의 성능을 위해 동시 다운로드 수(1-10)를 조정.
- **파일 중복 제거**: URL 해시를 사용하여 중복 다운로드 방지.
- **이미지 미리보기**: 콘텐츠를 확인하기 위해 다운로드 전에 이미지 미리보기.
- **상세 로깅**: 앱 내 콘솔로 진행 상황을 추적하고 문제를 해결.
- **크로스플랫폼 UI**: 여러 운영 체제와 호환되는 현대적이고 직관적인 인터페이스를 PyQt6로 구축.
- **미디어 재생**: 내장 재생 컨트롤(재생, 일시정지, 탐색, 볼륨)로 비디오와 GIF를 미리보기.
- **다국어 지원**: 영어, 일본어, 한국어, 중국어 간에 동적으로 전환.
- **자동 업데이트**: 선택적 알림과 함께 시작 시 새 버전 확인.

### Briefcase로 빌드하기 (모든 플랫폼)
1. 시스템에 **Python 3.9+**가 설치되어 있는지 확인하세요(Windows, macOS, Linux).
2. 이 저장소를 클론합니다:
   ```bash
   git clone https://github.com/VoxDroid/KemonoDownloader.git
   cd KemonoDownloader
   ```
3. 가상 환경을 만들고 활성화합니다:
    ```bash
    python -m venv .venv
    .venv/Scripts/activate # Windows 
    source .venv/bin/activate # Linux
    ```
4. 의존성을 설치합니다:
   ```bash
   python -m pip install -r requirements.txt
   ```
5. 애플리케이션을 빌드합니다:
   - **Windows**: `briefcase build windows`
   - **macOS**: `briefcase build macos`
   - **Linux**: `briefcase build linux`
   - **일반**: `briefcase build` *(권장)*
6. 애플리케이션을 실행합니다:
   - **Windows**: `briefcase run windows`
   - **macOS**: `briefcase run macos`
   - **Linux**: `briefcase run linux`
   - **일반**: `briefcase dev` *(권장)*
   - **참고**: Kemono.cr에서 콘텐츠를 가져오려면 인터넷 연결이 필요합니다.

## 사용법
시작하면 "실행" 버튼이 있는 소개 화면이 나타납니다. 클릭하여 **게시물 다운로더**, **크리에이터 다운로더**, **설정**, **도움말** 네 개의 탭이 있는 메인 인터페이스에 들어갑니다. 앱 내 도움말 탭에는 포괄적인 사용자 매뉴얼이 포함되어 있습니다.

### 시작하기
- 애플리케이션은 지정된 저장 위치에 기본 디렉토리(`Downloads`, `Cache`, `Other Files`)를 생성합니다.
- Kemono.cr 콘텐츠에 액세스하기 위해 활성 인터넷 연결을 확인하세요.
- 자세한 지침과 문제 해결 팁은 도움말 탭을 탐색하세요.

### 게시물 다운로더 탭
- **목적**: 개별 Kemono.cr 게시물에서 파일 다운로드.
- **사용법**:
  1. "게시물 URL 입력" 필드에 게시물 URL(예: `https://kemono.cr/patreon/user/123456789/post/123456789`)을 입력합니다.
  2. "대기열에 추가"를 클릭하여 목록에 추가합니다.
  3. 눈 아이콘을 클릭하여 파일을 보고, 유형별로 필터링하고(예: JPG, ZIP), 다운로드할 파일을 선택합니다.
  4. "다운로드"를 클릭하여 시작하고, 진행률 표시줄과 콘솔로 진행 상황을 모니터링합니다.

### 크리에이터 다운로더 탭
- **목적**: 크리에이터 프로필에서 콘텐츠를 일괄 다운로드.
- **사용법**:
  1. "크리에이터 URL 입력" 필드에 크리에이터 URL(예: `https://kemono.cr/patreon/user/123456789`)을 입력합니다.
  2. "대기열에 추가"를 클릭하여 목록에 추가합니다.
  3. 눈 아이콘을 클릭하여 게시물을 가져오고, 옵션(메인 파일, 첨부파일, 콘텐츠 이미지)을 구성하고, 게시물을 선택합니다.
  4. "다운로드"를 클릭하여 시작하고, 인터페이스를 통해 진행 상황을 추적합니다.

### 설정 탭
- **목적**: 애플리케이션의 동작 사용자 지정.
- **사용법**:
  1. 다운로드를 위한 폴더 이름과 저장 디렉토리를 설정합니다.
  2. 슬라이더나 스핀박스를 사용하여 동시 다운로드 수(1-10)를 조정합니다.
  3. "변경사항 적용"을 클릭하여 저장합니다.

## 릴리즈
- **Windows**: 미리 컴파일된 `.exe`가 [릴리즈 섹션](https://github.com/VoxDroid/KemonoDownloader/releases)에서 사용 가능.
- **macOS**: 미리 컴파일된 유니버설 `.dmg`(x86_64 및 Apple Silicon)가 [릴리즈 섹션](https://github.com/VoxDroid/KemonoDownloader/releases)에서 사용 가능.
- **Linux**: 미리 컴파일된 `.rpm`(Fedora/Red Hat용), `.deb`(Debian/Ubuntu용), 또는 `.pkg.tar.tsz`(Arch/Pacman용)가 [릴리즈 페이지](https://github.com/VoxDroid/KemonoDownloader/releases)에서 사용 가능.
- 새로운 기능, 버그 수정, 버전 업데이트에 대한 자세한 내용은 릴리즈 노트를 확인하세요.
- Briefcase로 빌드된 Python 소스는 적절한 설정으로 모든 플랫폼을 지원하는 주요 방법으로 남아있습니다.

## 지원
도움을 받거나, 문제를 신고하거나, 프로젝트 개발을 지원하는 방법에 대해서는 [지원 페이지](SUPPORT.md)를 참조하세요.

## 기여
Kemono Downloader는 오픈 소스이며, 기여를 환영합니다! 이슈나 풀 리퀘스트를 제출하기 전에 [기여 가이드라인](CONTRIBUTING.md), [행동 강령](CODE_OF_CONDUCT.md), [보안 정책](SECURITY.md)을 읽어주세요. 버그 신고, 기능 제안, 기타 기여에는 적절한 [이슈 템플릿](.github/ISSUE_TEMPLATE/)을 사용하고, 코드 제출에는 [풀 리퀘스트 템플릿](.github/PULL_REQUEST_TEMPLATE.md)을 사용하세요.

## 보안

이 프로젝트는 [MIT 라이선스](LICENSE) 하에 라이선스됩니다. 라이선스 조건에 따라 자유롭게 사용, 수정, 배포하세요.

## 의존성
소스에서 빌드하려면 다음 Python 패키지를 설치하세요:
- `PyQt6` (GUI용)
- `requests` (HTTP 요청용)
- `beautifulsoup4` (HTML 파싱용)
- `qtawesome` (아이콘용)
- `briefcase` (앱 패키징용)

이러한 의존성이 포함된 `requirements.txt` 파일을 만들고 `pip install -r requirements.txt`를 실행하세요.
