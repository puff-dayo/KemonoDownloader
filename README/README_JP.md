**Kemono Downloader**へようこそ。PyQt6を使用して構築された多機能なPythonベースのデスクトップアプリケーションで、[Kemono.cr](https://kemono.cr)からコンテンツをダウンロードするために設計されています。このツールにより、ユーザーはPatreon、Fanboxなどのサービスから個別の投稿またはクリエイター全体のプロフィールをアーカイブでき、カスタマイズ可能な設定と高度な機能を備えた幅広いファイルタイプをサポートします。

### 倫理的使用ガイドライン

KemonoDownloaderは、Patreon、Pixiv Fanbox、Gumroadなどの有料プラットフォームに元々投稿された素材を含む可能性があるKemono.crのコンテンツとやり取りします。これらのプラットフォームの多くのクリエイターは、生計を立てるために有料サブスクリプションに依存しています。許可なくそのコンテンツをダウンロードし再配布することは、彼らの創作活動を続ける能力を害する可能性があります。

ユーザーには以下を強く推奨します：
- KemonoDownloaderを責任を持って使用し、法的にアクセスする権利があるコンテンツのみに使用してください。

### リスクと制限

- **法的リスク**: Kemono.crからコンテンツをダウンロードすることは、著作権法または元のプラットフォームのサービス利用規約に違反する可能性があります。ユーザーはこのツールの使用に関連するすべての法的リスクを負うものとします。

## 機能
- **投稿ダウンロード**: URLを使用してKemono.crの特定の投稿からファイルを簡単にダウンロード。
- **クリエイターアーカイブ**: ワンクリックでクリエイターのプロフィールからすべての投稿とファイルを一括ダウンロード。
- **ファイルタイプサポート**: 画像（JPG、PNG、GIF）、動画（MP4）、アーカイブ（ZIP、7Z）、PDF、その他を処理。
- **並行ダウンロード**: 最適なパフォーマンスのために同時ダウンロード数（1-10）を調整。
- **ファイル重複除去**: URLハッシュを使用して冗長なダウンロードを防止。
- **画像プレビュー**: ダウンロード前に画像をプレビューしてコンテンツを確認。
- **詳細ログ**: アプリ内コンソールで進行状況を追跡し、問題をトラブルシューティング。
- **クロスプラットフォームUI**: 複数のオペレーティングシステムと互換性のある、モダンで直感的なインターフェースをPyQt6で構築。
- **メディア再生**: ビルトイン再生コントロール（再生、一時停止、シーク、音量）でビデオとGIFをプレビュー。
- **多言語サポート**: 英語、日本語、韓国語、簡体字中国語間で動的に切り替え。
- **自動更新**: 起動時に新しいバージョンをチェック、オプションの通知付き。

## インストール
Kemono Downloaderは現在、[Briefcase](https://briefcase.readthedocs.io/)を使用してパッケージ化されており、プラットフォーム間でネイティブアプリケーションとして実行または配布することが容易になっています。ソースからビルドするか、利用可能な場合は事前にコンパイルされたバイナリを使用できます。

### 事前にコンパイルされたバイナリ - **.EXE ファイル**
事前にコンパイルされたバイナリは、**Ko-fi**への貢献を通じて利用できます。ソースからビルドする代わりにすぐに実行できる実行ファイルの利便性をお求めの場合は、こちらでプロジェクトをサポートしてください：**[Ko-fiでバイナリを入手](https://ko-fi.com/izeno/shop)**  

#### 利用可能なバイナリ
- **Windows**:  
  - **ポータブル**: `W-KemonoDownloader-x86_64-Portable.exe`  
  - **インストーラー**: `W-KemonoDownloader-x86_64.msi`  
- **macOS**:  
  - `M-KemonoDownloader-arm64.dmg` (Apple Silicon)  
  - `M-KemonoDownloader-x86_64.dmg` (Intel)  
- **Linux**:  
  - `L-KemonoDownloader-x86_64.pkg.tar.zst` (Archベース)  
  - `L-KemonoDownloader-x86_64.rpm` (Fedora/Red Hatベース)  
  - `L-KemonoDownloader-amd64.deb` (Debian/Ubuntuベース)  

継続的な開発をサポートし、こちらからアクセスしてください：**[Ko-fi バイナリページ](https://ko-fi.com/izeno/shop)**

### Briefcaseでのビルド（全プラットフォーム）
1. システムに**Python 3.9+**がインストールされていることを確認してください（Windows、macOS、Linux）。
2. このリポジトリをクローンします：
   ```bash
   git clone https://github.com/VoxDroid/KemonoDownloader.git
   cd KemonoDownloader
   ```
3. 仮想環境を作成してアクティブ化します：
    ```bash
    python -m venv .venv
    .venv/Scripts/activate # Windows 
    source .venv/bin/activate # Linux
    ```
4. 依存関係をインストールします：
   ```bash
   python -m pip install -r requirements.txt
   ```
5. アプリケーションをビルドします：
   - **Windows**: `briefcase build windows`
   - **macOS**: `briefcase build macos`
   - **Linux**: `briefcase build linux`
   - **一般**: `briefcase build` *（推奨）*
6. アプリケーションを実行します：
   - **Windows**: `briefcase run windows`
   - **macOS**: `briefcase run macos`
   - **Linux**: `briefcase run linux`
   - **一般**: `briefcase dev` *（推奨）*
   - **注意**: Kemono.crからコンテンツを取得するためにはインターネット接続が必要です。

## 使用方法
起動すると、「起動」ボタンがある導入画面が表示されます。クリックしてメインインターフェースに入ると、**投稿ダウンローダー**、**クリエイターダウンローダー**、**設定**、**ヘルプ**の4つのタブがあります。アプリ内のヘルプタブには包括的なユーザーマニュアルが含まれています。

### はじめに
- アプリケーションは、指定された保存場所にデフォルトディレクトリ（`Downloads`、`Cache`、`Other Files`）を作成します。
- Kemono.crのコンテンツにアクセスするために、アクティブなインターネット接続を確保してください。
- 詳細な手順とトラブルシューティングのヒントについては、ヘルプタブを確認してください。

### 投稿ダウンローダータブ
- **目的**: 個別のKemono.cr投稿からファイルをダウンロード。
- **使用方法**:
  1. 「投稿URLを入力」フィールドに投稿URL（例：`https://kemono.cr/patreon/user/123456789/post/123456789`）を入力します。
  2. 「キューに追加」をクリックしてリストに追加します。
  3. 目のアイコンをクリックしてファイルを表示し、タイプでフィルター（例：JPG、ZIP）し、ダウンロードするファイルを選択します。
  4. 「ダウンロード」をクリックして開始し、プログレスバーとコンソールで進行状況を監視します。

### クリエイターダウンローダータブ
- **目的**: クリエイターのプロフィールからコンテンツを一括ダウンロード。
- **使用方法**:
  1. 「クリエイターURLを入力」フィールドにクリエイターURL（例：`https://kemono.cr/patreon/user/123456789`）を入力します。
  2. 「キューに追加」をクリックしてリストに追加します。
  3. 目のアイコンをクリックして投稿を取得し、オプション（メインファイル、添付ファイル、コンテンツ画像）を設定し、投稿を選択します。
  4. 「ダウンロード」をクリックして開始し、インターフェースで進行状況を追跡します。

### 設定タブ
- **目的**: アプリケーションの動作をカスタマイズ。
- **使用方法**:
  1. ダウンロード用のフォルダー名と保存ディレクトリを設定します。
  2. スライダーまたはスピンボックスを使用して同時ダウンロード数（1-10）を調整します。
  3. 「変更を適用」をクリックして保存します。

### ヘルプタブ
- **目的**: 埋め込まれたユーザーマニュアルにアクセス。
- **使用方法**: ヘルプタブに移動して、詳細なガイド、例、サポート情報を読むことができます。
